# Building TaskWarrior from Source

If your operating system doesn't provide a TaskWarrior 3.x binary, you'll need to build it from source.

## Prerequisites

- **CMake** (version 3.10 or higher)
- **Build tools** (gcc, make, etc.)
- **Development libraries**: libgnutls28-dev, uuid-dev
- **Rust** (for building TaskWarrior)
